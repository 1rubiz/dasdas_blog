## Installation

clone the repo
```bash
    git clone https://github.com/1rubiz/dasdas_blog.git
    
    cd dasdas_blog
```
Install project dependencies with npm

```bash
  npm install
```
    
## Deployment

To deploy this project run

```bash
  npm run dev
```

To build this project run
```bash
  npm run build
```
